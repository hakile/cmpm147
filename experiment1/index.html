Experiment 1 - Hello Glitch/Generator

Description

Remix a generator on glitch.com and integrate it into my portfolio site.

Technical

A grammar of strings is defined by replacement tokens, which gets replaced by a random word each time it appears in the text.

Reflection

The hardest parts of the experiment were coming up with multiple possibilities for each token so the experience wouldn't feel stale, and integrating the code on Glitch into the portfolio site.
